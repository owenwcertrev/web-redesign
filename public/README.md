# SEO Images - TODO

## Required Images for Optimal SEO

### 1. Open Graph Image (1200x630px)
**File:** `og-image.png`
- Used for Facebook, LinkedIn, and other social platforms
- Should include CertREV logo and tagline
- Background should be brand colors (beige, navy, coral, lime)
- High contrast text for readability

### 2. Twitter Card Image (1200x600px)
**File:** `twitter-image.png`
- Used specifically for Twitter/X
- Similar to OG image but with 2:1 aspect ratio
- Can be same as OG image, Twitter will crop

### 3. Favicon (32x32px)
**File:** `favicon.ico`
- Small icon for browser tabs
- Should be simple, recognizable version of logo
- Works at very small sizes

### 4. Apple Touch Icon (180x180px)
**File:** `apple-touch-icon.png`
- Used when users add site to iOS home screen
- PNG format with rounded corners
- Background color recommended (no transparency)

### 5. Logo (Recommended: SVG or high-res PNG)
**File:** `logo.png` or `logo.svg`
- Used in Organization schema
- Used for structured data
- Ideally square (1:1 ratio), at least 600x600px
